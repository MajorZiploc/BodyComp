# Body Comp

## Requirements
- python3.8
- pip

## Development tools
- vscode
- python extension for vscode
- bash
- wsl ubuntu

## Setup

# first time up front setup
> mkdir ~/.virtualenvs
> python -m venv ~/.virtualenvs/body_comp_server

# launch virtual environment
> source ~/.virtualenvs/body_comp_server/bin/activate

# install dependencies
> pip install -e .
